This repository contains my work completed during the LogicStack Data Analytics Internship. The projects focus on analyzing retail sales data using Google Sheets, Python, Pandas, and Microsoft Power BI to develop practical data analysis and visualization skills.

📅 Week 1 – Retail Sales Excel Analysis
📖 Project Overview

This project focused on analyzing a retail sales dataset using Google Sheets (Excel) and Python. The objective was to clean and validate the dataset, perform basic business analysis, create visualizations, and document meaningful insights.

🎯 Project Objectives
Understand the structure of a retail sales dataset.
Clean and organize the data for analysis.
Perform data quality checks.
Calculate key business metrics.
Analyze sales by product category and gender.
Create charts to visualize business insights.
Document observations based on the analysis.
🛠️ Tools & Technologies
Google Sheets (Excel)
Python 3
Jupyter Notebook
Pandas
GitHub
Ubuntu Linux
📂 Dataset Information

The dataset contains 1,000 retail transactions with the following columns:

Transaction ID
Date
Customer ID
Gender
Age
Product Category
Quantity
Price per Unit
Total Amount
✅ Tasks Completed
1. Dataset Understanding
Explored the dataset structure.
Identified rows, columns, and data types.
2. Data Formatting
Formatted the dataset.
Adjusted column widths.
Applied date and number formatting.
Froze the header row.
3. Data Quality Checks
Checked for blank values.
Checked for duplicate records.
Verified quantity values.
Verified price values.
Validated Total Amount calculations.
4. Basic Analysis

Calculated:

Total Sales
Average Sale
Minimum Sale
Maximum Sale
Total Quantity Sold
Average Customer Age
Youngest Customer
Oldest Customer
Total Transactions
Unique Product Categories
5. Sorting & Filtering
Identified highest and lowest transactions.
Filtered records by:
Product Category
Gender
Quantity
Sorted data by sales and customer age.
6. Category-Based Analysis
Total Sales by Product Category
Quantity Sold by Product Category
Total Sales by Gender
7. Charts

Created:

📊 Bar Chart – Total Sales by Product Category
📈 Column Chart – Quantity Sold by Product Category
🥧 Pie Chart – Total Sales by Gender
8. Observations

Documented key insights based on the analysis.

📌 Key Observations
Clothing products recorded the highest quantity sold, followed by Electronics and Beauty.
Electronics generated the highest total sales.
Most customers in the dataset were female.
A total of 2,514 products were sold across 1,000 transactions.
Customer ages ranged from 18 to 64 years, with an average age of approximately 41 years.
No missing values, duplicate records, or calculation errors were found during the data quality checks.
